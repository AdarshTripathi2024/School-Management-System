# School Management System (Laravel)

## Overview
This is a web-based School Management System developed using the Laravel PHP framework. 
The system provides separate dashboards and login access for Admin, Teacher, Student, 
and Parent roles.

## Features

### Admin Module
- Add and manage students, teachers, and parents
- Assign classes, subjects, class teachers, and subject teachers
- Daily attendance management
- Fee management system
- Upload student results
- Manage complaints and notices
- Manage Holidays
- Make DB Backup
- Manage Software Runs Fast by removing the old logs & cache related to routes etc.

### Teacher Module
- View assigned classes and subjects he/she has to teach in each classes.
- Mark daily attendance of the class where she/he is class teacher
- View and respond to complaints in which he/she mentioned by the parents
- Upload student results of his own class.
- Access Notices & Holidays 

### Student Module

- Access notices & holidays
